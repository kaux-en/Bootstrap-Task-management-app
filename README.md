Welcome to my Task Management App!
This application was completed using HTML Bootstrap and includes fun forms, pretty pictures and interesting interactions. (see what I did there)
What you will see in this file is all Bootstrap material.
It starts first with a Navigation bar attached to the landing page picture.
Then we go into the pages; the Homepage, Task Dashboard, Task Details, Task Creation, Register page and the Authentication page.
These pages are also separated in the html file by their specific page titles that are commented out. 
The homepage presents the Task Managers features and details of each feature.
Your Task Dashboard showcases the tasks that you have currently with a badge that tells you how many tasks that you have due that day.
Task Details allows you to change an already existing task, while Task Creation allows you to create a new Task.
If you're a new user, the Register page is where you would go to sign up for the app.
Lastly, you'd authenticate your new username and password with the authentication page. 
I hope you enjoy my Task Management Application!
